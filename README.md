# 🎧 AudioHeavy

**Tropical house DJ mixes, music, and merch.**

Live site: [audioheavy.com](https://audioheavy.com)

## About

AudioHeavy is a personal music brand and portfolio site for DJ mixes, primarily in the tropical house genre. The site promotes new mixes released on [Mixcloud](https://www.mixcloud.com/audioheavymusic/) and [YouTube](https://www.youtube.com/@audioheavy), and will eventually offer merchandise.

This project also serves as a hands-on learning experience in modern web development, Python APIs, AI tooling, and cloud deployment.

## Tech Stack

| Layer | Technology | Status |

| Frontend | HTML / CSS / JS | ✅ Live |

## Development

This site is built with AI-assisted development using [Warp](https://www.warp.dev/) as the primary development environment.

```bash
# Clone the repo
git clone https://github.com/luiseduardocolon/audioheavy.git
cd audioheavy

# Phase 1 is static HTML — just open index.html in a browser
open index.html
```
